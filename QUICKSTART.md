# Quick Start Guide

## Installation Steps

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser at http://localhost:3000

## What's Included

✅ Home page with hero section and features
✅ About page with mission, vision, and domains
✅ Events listing page (upcoming and past)
✅ Dynamic event detail pages
✅ Registration form with validation
✅ Contact page with form
✅ Admin panel for event management
✅ Dark mode support
✅ Responsive design
✅ API routes for forms
✅ MongoDB integration ready

## Next Steps

1. Customize event data in `src/lib/events.ts`
2. Add your MongoDB URI to `.env` file
3. Update social media links in Footer component
4. Add email service for confirmations
5. Deploy to Vercel or your preferred platform
